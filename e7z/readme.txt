Easy 7-Zip 0.1.6
----------------
Portions Copyright (C) 2013-2016 James Hoo

Easy 7-Zip is an easy-to-use version of 7-Zip. Easy 7-Zip was built based on 7-Zip 15.06 beta. 
Easy 7-Zip keeps all features of 7-Zip and adds a few useful features that makes 7-Zip more
user-friendly.


Version 0.1
1. Adds icons to context menu
2. Adds options to Extract dialog
3. Keeps same output folder history
4. Minimizes to system tray when clicking "Background" on progress dialog
5. Makes a new installation file
6. Fixes bugs

Version 0.1.2
1. Adds option "Delete source archive after extraction"
2. Adds option "Close 7-Zip after extraction"
3. Save option "Show password" on pop extracting dialog box

Version 0.1.3
1. Moves codes to 7-Zip 9.35 beta

Version 0.1.4
1. Moves codes to 7-Zip 9.38 beta

Version 0.1.5
1. Moves codes to 7-Zip 15.06 beta

Version 0.1.6
1. Moves codes to 7-Zip 16.04


Easy 7-Zip is compatible with Windows 10/8.1/8/7/Vista/2012/2008/2003/XP/2000.
(both 32-bit and 64-bit compatible)

Please contact me at <james at e7z dot org> if you have questions or suggestions with 
Easy 7-Zip.

7-Zip 16.04
-----------

7-Zip is a file archiver for Windows NT / 2000 / 2003 / 2008 / 2012 / XP / Vista / 7 / 8 / 10. 

7-Zip Copyright (C) 1999-2016 Igor Pavlov.

The main features of 7-Zip: 

  - High compression ratio in the new 7z format
  - Supported formats:
     - Packing / unpacking: 7z, XZ, BZIP2, GZIP, TAR, ZIP and WIM.
     - Unpacking only: AR, ARJ, CAB, CHM, CPIO, CramFS, DMG, EXT, FAT, GPT, HFS,
                       IHEX, ISO, LZH, LZMA, MBR, MSI, NSIS, NTFS, QCOW2, RAR, 
                       RPM, SquashFS, UDF, UEFI, VDI, VHD, VMDK, WIM, XAR and Z.
  - Fast compression and decompression
  - Self-extracting capability for 7z format
  - Strong AES-256 encryption in 7z and ZIP formats
  - Integration with Windows Shell
  - Powerful File Manager
  - Powerful command line version
  - Localizations for 85 languages
